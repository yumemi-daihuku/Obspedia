📖 このサイトについて

Obspedia とは

Obspedia（オブスペディア）は、性に多感なドスケベクソガキの為に暇を持て余したドスケベが作ったドスケベの為のドスケベ百科です。性への知識を渇望する子供達に引導を渡すために作成しました。
英語で「わいせつな」を意味する「obscene」と、「pedia」がサイト名の由来です。 記事は全てドスケベな内容でなくてはなりません。性欲に忠実に生きましょう。

運営について

このサイトはカイロスが運営しています。
記事の内容はユーザーによって自由に投稿・編集されるものであり、 運営者がすべての内容を監修しているわけではありません。が、私がドスケベでないと感じた記事は削除させて頂きます。

使用技術

Obspedia は以下の技術を使って構築されています。

・フロントエンド: HTML / CSS / JavaScript（バニラ）
・データベース: Firebase Realtime Database
・ホスティング: GitHub Pages ・サポート: Claude
